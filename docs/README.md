# 介绍

`react-enahcned` 是一套无侵入 `react` 架构增强器，封装了 `model-redux`，`api-manage` 等以及内置多种集成开发组件（antd 组件、自定义组件）以及实用工具，简化开发流程、降低 `react` 开发门槛， 实现快速开发

## 设计思想

-   降低 react 开发门槛，以及提高开发效率
-   无侵入使用，对已使用的框架友好，并且维护增加一些辅助开发的功能
